## RV32I

### Instruction Types

| Type | Description               |
|------|---------------------------|
| R    | Register to register      |
| I    | Short immediate and loads |
| S    | Stores                    |
| U    | Long immediate (Unique)   |
| B    | Branches (conditional)    |
| J    | Jumps (unconditional)     |
